# RFLink Raw Tools v0.0.1

First public-ready HACS baseline.

## Highlights

- Sidebar app for RFLink Raw Tools.
- Send learned RFLink commands.
- Captured page for RFLink entities and recent RFLink log lines.
- Debug page with decoded/raw RF logging switches.
- Setup page that can detect/install a conservative `rflink:` YAML block if missing.
- HACS and Hassfest validation workflows added.
