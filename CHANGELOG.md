# Changelog

## v0.0.1

Initial public-ready baseline for RFLink Raw Tools.

- Sidebar app for RFLink command sending and capture support.
- Send learned RFLink commands.
- Show RFLink-discovered entities and recent RFLink log lines.
- Debug controls for decoded RFLink logging and raw RF capture logging.
- Setup helper to detect RFLink and add a conservative `rflink:` YAML block when missing.
- HACS metadata and validation workflows.
